# Riding Cities 🛹

Site vitrine d’une association de skate mettant en avant les activités, les membres fondateurs et les plannings de cours.
<img width="960" height="504" alt="image" src="https://github.com/user-attachments/assets/0ff7cd40-6efc-4f0a-8524-9936b5fc2875" />

## Features

- Ajout d’une section “Membres fondateurs”
- Intégration des photos et noms des fondateurs
- Mise à jour de la section planning
- Boutons de téléchargement des plannings
- Intégration fidèle à une maquette

## Tech Stack

- HTML5
- CSS3
- Responsive Design
- Figma
- GitHub Pages
